# 🐾 PetCare

Plataforma web tipo Rappi para servicios de mascotas 🐶🐱

## 🚀 Funcionalidades

- Registro y login de usuarios
- Registro de empresas veterinarias
- Gestión de mascotas
- Publicación de servicios
- Agendamiento de citas
- Confirmación y cancelación de citas

## 🛠 Tecnologías

### Frontend
- React (Vite)
- JavaScript
- CSS

### Backend
- Node.js
- Express

### Base de datos
- PostgreSQL (Supabase)

## 📂 Estructura

petcare/
│── backend/
│── frontend/

### Backend
